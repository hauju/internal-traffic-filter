# Privacy Policy

**Internal Traffic Filter** is committed to protecting your privacy.

## Data Collection

This extension **does not collect, store, or transmit any personal data** to external servers.

## Data Storage

The extension stores the following data locally in your browser:

- **Blocked domains list**: The list of websites where you have chosen to filter your traffic. This data is stored using Chrome's sync storage, which may sync across your devices if you are signed into Chrome/Brave with sync enabled.

## Data Usage

- Your blocked domains list is used solely to determine whether to set localStorage flags on websites you visit.
- No analytics, tracking, or telemetry is performed by this extension.
- No data is shared with third parties.

## Permissions

The extension requires the following permissions:

- **storage**: To save your list of blocked domains
- **activeTab**: To interact with the current tab when you click the extension
- **scripting**: To set localStorage flags on blocked websites
- **host_permissions (all URLs)**: To run on any website where you may want to filter your traffic

## Local Operation

All extension functionality operates entirely within your browser. The extension:

- Does not make any network requests
- Does not have a backend server
- Does not collect usage statistics

## Changes to This Policy

Any changes to this privacy policy will be reflected in this document.

## Contact

If you have questions about this privacy policy, please [open an issue](https://github.com/hauju/internal-traffic-filter/issues) on GitHub.
